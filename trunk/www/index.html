accurev4idea

Welcome to the home page of accurev4idea IntelliJ IDEA plugin!

What's currently included?

  • Depots / Streams tree widget
  • Diff against backed version
  • Diff against any previous transaction / revision

    What's coming?

  • AccuRev project Wizard - ability to create a workspace from any stream and open it as Idea project
  • AccuRev module Wizard - ability to add module to existing project directly from AccuRev
  • Full refactoring support, add, delete, move, rename etc
  • Full transactional support for "keep" and "promote" operations
  • Full support for auxiliary tasks such as update, pop etc
  • Ability to "send to Issue" on project commit
  • More features as we find needed.

    Which version of Idea is supported?

  • Currently we support 4.5+ version (build 2253). Version for 5.0 is planned for later, once it is officially released.

    How do I install this plugin?

  • Please head over to IntelliJ, open settings menu, go to Plugins section and click on Available Plugins tab. After pane is refreshed you should be able to see "accurev4idea" plugin. If you also see "AccuRev" plugin right above please note that this is not a plugin written by us.
